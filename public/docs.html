Using the home mode.
You type stuff in that one box and hit enter. You can select the box using CTRl+E.

Those other buttons do stuff to.

Fullscreen will toggle fullscreen mode. CTRL+F will toggle this button.

Settings will toggle some global settings. CTRL+S will toggle this button.

Docs will toggle the documentation. CTRL+D will toggle this button.

Mode will toggle the mode dropdown. CTRL+M will toggle this button.
